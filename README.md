# HelloCompose

## Опис
Простий Android-застосунок, створений з використанням Jetpack Compose.

## Функціональність
- Відображення тексту "Hello Android"

## Технології
- Kotlin
- Jetpack Compose
- Android Studio

## Інструкція запуску
1. Відкрити проект в Android Studio
2. Запустити емулятор
3. Натиснути Run
